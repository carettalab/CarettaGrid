This page demonstrates various techniques for optimizing DataView performance for large client-side datasets. This page displays an interactive grid with 500'000 rows with real-time filtering.

This is achieved by:

Inlining filter function to cut down on the cost of function calls Providing hints to indicate whether a filtering operation will result in narrowing or expanding scope or whether the scope is unchanged. Providing a range of rows for which onRowsChanged even should be fired.
